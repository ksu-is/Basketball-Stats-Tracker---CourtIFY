# Project Roadmap - Basketball Stats Tracker

## Overview
This project is a web-based Basketball Stats Tracker built using Flask and SQLite. The application allows users to add players, record game statistics, and view player performance over time.

## Sprint 1 (Planning & Setup)
- Brainstormed project idea based on real-life basketball stat tracking
- Set up Flask application
- Created initial project structure
- Established database connection

## Sprint 2 (Core Development)
- Created player database table
- Built routes for adding players
- Designed main dashboard page
- Implemented templates for user interface


## Future Improvements
- Add player editing and deletion
- Add player averages and advanced stats
- Improve UI design
- Add user authentication

## Sprint 3 (Feature Expansion)
- Added game statistics tracking
- Created player detail pages
- Implemented total points calculations
- Improved UI with CSS styling

## Sprint 4 (Debugging & Deployment)
- Fixed TemplateNotFound errors
- Organized project folders correctly (templates/static)
- Tested application locally
- Uploaded project to GitHub


## Reflection
This project helped me understand how Flask connects backend logic with frontend templates and how proper file structure is critical for web applications. I also learned how to debug common errors and organize a full-stack application.
