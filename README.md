# Basketball Stats Tracker

## Overview
Basketball Stats Tracker is a web-based application built using Flask and SQLite that allows users to manage basketball players and track their game-by-game performance.

## Features
- Add and manage players
- Record game statistics
- View total season performance
- Automatically calculate total points
- Store notes for each game

## Technologies Used
- Python
- Flask
- SQLite
- HTML/CSS

## How to Run the App

1. Install Flask:
   pip install flask

2. Navigate to the project folder

3. Run the application:
   python app.py

4. Open your browser and go to:
   http://127.0.0.1:5000

## Stats Tracked
- Free throws
- Steals
- 2-point makes
- 3-point makes
- Turnovers
- Rebounds
- Assists
- Fouls

## Purpose
This application helps coaches, players, or teams track performance data and analyze player contributions over time.
